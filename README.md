## 📝 Описание проекта

Проект **"Media"** представляет собой ленту постов (timeline) с возможностью прикрепления геопозиции к каждому сообщению. Пользователь может создавать текстовые посты, которые автоматически привязываются к его текущему местоположению через Geolocation API.

### ✨ Основные возможности

- 📝 Создание текстовых постов
- 📍 Автоматическое определение геопозиции
- ⌨️ Ручной ввод координат при ошибке геолокации
- 🎨 Современный адаптивный дизайн
- ✅ Валидация вводимых координат
- 🔄 Автоматическая сборка и деплой через GitHub Actions

### 🔧 Технологии

- **Webpack** - сборка проекта
- **Jest** - тестирование
- **GitHub Actions** - CI/CD
- **Geolocation API** - определение местоположения
- **HTML5/CSS3** - интерфейс
- **JavaScript (ES6+)** - логика приложения

## 📋 Функциональные требования

### Основной функционал
- При вводе текста и нажатии Enter создается новый пост
- Автоматический запрос геолокации при создании поста
- Если геолокация недоступна - появляется модальное окно для ручного ввода координат

### Валидация координат
Функция парсинга координат обрабатывает следующие форматы:
- `51.50851, −0.12572` (с пробелом)
- `51.50851,−0.12572` (без пробела)
- `[51.50851, −0.12572]` (с квадратными скобками)

При неверном формате генерируется исключение с соответствующим сообщением.

## 🚀 Установка и запуск

### Предварительные требования
- Node.js версии 18 или выше
- npm (менеджер пакетов)

### Инструкция по установке

```bash
# Клонирование репозитория
git clone https://github.com/lokilaufer/ahj-media.git

# Переход в директорию проекта
cd ahj-media

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm start

# Сборка для продакшена
npm run build

# Запуск тестов
npm test

# Просмотр покрытия тестами
npm run coverage

# Деплой на GitHub Pages
npm run deploy
